BuddyBeats

BuddyBeats is an application to help you make YouTube and Spotify playlists with your friends, helpful for everything from deciding on the music at your wedding to ensuring crowd-pleasing hits at your holiday party.

This was my final project for GA and it was truly fun to make. The core functionality took less time than I expected, so I had more time than usual to tweak and optimize the site, adding in new features and playing around. In fact, I had never intended to add Spotify at all.

I am really proud of how it turned out. It is an easy-to-use application and for a YouTube and playlist addict like myself, it provides a great tool that YouTube itself does not provide.

Technologies Used:

  • Active Record
  • Bootstrap
  • HTML 5
  • Javascript
  • jQuery
  • PostgreSQL
  • Rake
  • Ruby on Rails
  • Underscore

CopyChef

CopyChef is a site for gathering copycat recipes of the dishes from your favorite restaurants.

My biggest challenge on this project was the search bar, but it paid off. The search utilizes auto-complete and searches by three fields (restaurant, name of dish and tags), while avoiding duplicate listings. For example, if you type in Panera, auto-complete will not suggest Panera three times for each recipe.

Technologies Used:

  • Backbone
  • Express
  • Handlebars
  • jQuery
  • jQuery UI | Autocomplete
  • MongoDB
  • Passport
  • Underscore
